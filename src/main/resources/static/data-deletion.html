Suppression des données utilisateur

Conformément à notre politique de confidentialité, les utilisateurs peuvent demander la suppression de leurs données personnelles à tout moment.

Comment faire une demande de suppression :

  • Envoyez un e-mail à contact@mediplan.com avec l’objet : “Suppression de mes données Facebook”.
  • Ou bien, si vous avez utilisé Facebook Login, vous pouvez supprimer vos données via votre compte Facebook en suivant : Paramètres des applications Facebook.

Une fois la demande reçue, vos données seront supprimées de nos serveurs dans un délai de 30 jours ouvrables.
